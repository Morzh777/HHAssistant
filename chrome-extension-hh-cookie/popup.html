Автоматическое сохранение:
При переходе на страницу вакансии информация автоматически сохраняется на бэкенд в папку data/vacancies/

Сохранение резюме:
Зайдите на hh.ru, авторизуйтесь, откройте свое резюме, затем нажмите кнопку выше для сохранения резюме на бэкенд.

Сброс резюме:
Удаляет все сохраненные файлы резюме для обновления данных.
